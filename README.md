# ⚡ Urja Insight: Electricity Demand Forecasting & Energy Analytics Platform

## 📖 Project Overview

Urja Insight is an end to end Data Analytics and Machine Learning project designed to analyze electricity demand and supply patterns, generate business insights, and forecast future energy requirements.

The project combines multiple technologies including Python, SQL, Power BI, Tableau, Hadoop, Spark, and Streamlit to create a comprehensive energy analytics platform.

## 🎯 Objectives

* ⚡ Analyze electricity demand and supply trends
* 📈 Forecast future energy requirements
* 🔍 Identify consumption patterns
* 📊 Create interactive dashboards
* 🤖 Build machine learning forecasting models
* ☁️ Demonstrate Big Data processing using Hadoop and Spark

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/rinshadrin/Urja-Insight.git
```

### Install Dependencies

```bash
pip install pandas numpy scikit-learn streamlit openpyxl
```

### Run Streamlit App

```bash
streamlit run app.py
```
